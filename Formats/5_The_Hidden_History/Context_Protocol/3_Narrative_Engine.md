# 3. Narrative Engine

## Story Structure: Mystery Reveal Framework
*   **The Hook (0:00-1:00):** Present bizarre fact with no context
*   **The Setup (1:00-3:00):** Historical background
*   **The Event (3:00-10:00):** Full story unfolds
*   **Why It Was Forgotten (10:00-12:00):** Suppression/obscurity reasons
*   **The Legacy (12:00-14:00):** Modern implications

## Episode Arc Pattern
*   Start with unbelievable claim
*   Provide evidence it's true
*   Tell the full story
*   Connect to present day

## Series Arc Potential
*   Geographic series: "Forgotten Americas", "Lost Europe"
*   Thematic: "Suppressed Science", "Banned Art"

## Character Role Definitions
*   **The Discoverer (narrator):** Uncovering secrets
*   **Historical Figures (characters):** Brought to life through animation
*   **The Establishment (antagonist):** Forces hiding the story
