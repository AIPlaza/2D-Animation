# 4. Visual Identity

## Animation Style
*   Hybrid archival + paint animation
*   Actual historical photos/documents
*   Animated reenactments over photos
*   Sepia tones transitioning to color for impact
*   Archive aesthetic (film grain, aged paper)

## Signature Visual Elements
*   "Classified" file folder opening
*   Animated historical documents
*   Map overlays showing locations
*   Before/after comparisons

## Color Strategy
*   **Historical:** Sepia, black & white
*   **Reveal moments:** Burst of color
*   **Modern connections:** Full color

## Motion Design Principles
*   Ken Burns effect on photos
*   Document animations (papers shuffling)
*   Time-period appropriate transitions
*   Dramatic reveals (fade from black)
