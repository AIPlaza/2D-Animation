# 4. Visual Identity

## Animation Style
*   Portrait-focused illustration
*   Detailed character art (vs simplified)
*   Expressive faces showing emotion
*   Split-screen showing public vs private
*   Symbolic imagery for psychology

## Signature Visual Elements
*   Portrait gallery intro
*   Thought bubble visualizations
*   Timeline of key events
*   Symbolic objects representing traits

## Color Strategy
*   **Subject's color:** Evolves through story
*   **Background:** Shifts with emotional tone
*   **Highlight color:** For key revelations

## Motion Design Principles
*   Close-ups on facial expressions
*   Zoom into eyes for intensity
*   Memory flashback transitions
*   Symbolic montages
