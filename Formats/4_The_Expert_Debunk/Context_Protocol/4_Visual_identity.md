# 4. Visual Identity

## Animation Style
*   Clean educational + demonstrative
*   Diagrammatic explanations
*   Before/after comparisons
*   Animated experiments
*   Highlighted contradictions

## Signature Visual Elements
*   "MYTH" stamp on claims
*   Scientific equipment aesthetics
*   Checkmarks (✓) and X marks (✗)
*   Side-by-side comparisons

## Color Strategy
*   **Myth state:** Hazy, uncertain (desaturated)
*   **Evidence phase:** Sharp, clear (high saturation)
*   **Verdict:** Green (confirmed) or Red (busted)

## Motion Design Principles
*   Slow, methodical reveals
*   Step-by-step animations
*   Rewind/replay for emphasis
*   Explosion effects for dramatic busts
