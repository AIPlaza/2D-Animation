# 3. Narrative Engine

## Story Structure: System Breakdown
*   **The Surface (0:00-1:30):** What people see/experience
*   **The Mechanism (1:30-7:00):** How it actually works
*   **The Exploitation (7:00-10:00):** How people game it
*   **The Consequences (10:00-12:00):** Intended and unintended effects

## Episode Arc Pattern
*   Start with relatable experience
*   Zoom out to system view
*   Show hidden mechanics
*   Explain implications

## Series Arc Potential
*   "How [Platform] Really Works"
*   "The Hidden Rules of [System]"
*   Multi-part deep dives on complex systems

## Character Role Definitions
*   **The User (everyman character):** Experiencing the system
*   **The System (visualized as machine):** The algorithm itself
*   **The Designer (occasional):** Intent vs reality
