# 4. Visual Identity

## Animation Style
*   Kinetic infographic + paint style
*   Constant movement
*   Time stamps on every scene
*   News ticker aesthetic
*   Quick cuts between events

## Signature Visual Elements
*   Omnipresent digital clock
*   Event ticker (bottom of screen)
*   Map showing simultaneous events
*   Domino effect visualizations

## Color Strategy
*   **Early hours:** Calm blues → 
*   **Peak intensity:** Reds/oranges → 
*   **Resolution:** Return to blue or fade to black

## Motion Design Principles
*   Speed ramping (slow → fast → slow)
*   Jump cuts on timeline
*   Zoom transitions between events
*   Clock ticking sound design
